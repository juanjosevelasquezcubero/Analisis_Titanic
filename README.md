# Análise Exploratória de Dados - Titanic

Este projeto tem como objetivo realizar uma Análise Exploratória de Dados (AED) utilizando a base de dados pública do Titanic. A análise foi desenvolvida em Python utilizando bibliotecas amplamente utilizadas na área de ciência de dados, como pandas, matplotlib e seaborn.

Inicialmente foi realizada a importação do conjunto de dados em formato CSV. Em seguida foram analisadas as características gerais do dataset, incluindo tipos de dados, estatísticas descritivas e identificação de valores nulos.

Durante a etapa de preparação dos dados foram tratados valores ausentes presentes em algumas colunas, como idade e porto de embarque, além da remoção de possíveis registros duplicados. Após a limpeza dos dados foi possível iniciar a exploração das informações.

Foram aplicadas técnicas de filtragem, ordenação e agrupamento de dados utilizando operações como groupby, permitindo analisar padrões relevantes entre variáveis como sexo, classe do passageiro e idade.

Também foram construídas visualizações gráficas para facilitar a interpretação dos dados, representando distribuições e relações entre variáveis importantes para compreender os fatores associados à sobrevivência dos passageiros.

A análise revelou alguns padrões relevantes, como maior taxa de sobrevivência entre mulheres e passageiros da primeira classe, além de diferenças relacionadas à idade e à tarifa paga pelos passageiros.

Esse tipo de análise exploratória é essencial no processo de ciência de dados, pois permite compreender melhor o comportamento dos dados antes da aplicação de modelos mais avançados de análise ou aprendizado de máquina.
